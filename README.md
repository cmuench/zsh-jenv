# zsh-jenv
A zsh plugin for the jenv Line Interface (jenv CLI) completions.

## Install
Using the [:zap: Zap](https://www.zapzsh.org/) minimal zsh plugin manager, add the following to your `.zshrc`

```sh
# Install plugin
plug "cmuench/zsh-jenv"
```
